# Dashboard_Streamlit - 量化交易看板

整合长线和短线策略结果的可视化看板。

## 功能特性

- **市场状态总览**: 实时显示当前市场状态 (激进/中性/防御)
- **长线配置展示**: 饼图展示资产配置权重
- **短线信号展示**: 热点板块、操作信号、龙头股
- **综合交易建议**: 仓位建议、板块偏好
- **快捷操作**: 一键刷新、运行策略入口

## 快速开始

```bash
cd Dashboard_Streamlit
streamlit run app.py
```

## 设计理念

采用 **解耦架构**:

```
                    +------------------+
                    |   Dashboard      |
                    |   (Streamlit)    |
                    +--------+---------+
                             |
                    +--------v---------+
                    |   DataBridge     |  <-- 统一数据访问
                    +--------+---------+
                             |
          +------------------+------------------+
          |                  |                  |
+---------v------+   +-------v------+   +------v-------+
| LongTerm       |   | ShortTerm    |   | MarketRegime |
| (权重/报告)     |   | (信号/热点)   |   | (市场状态)    |
+----------------+   +--------------+   +--------------+
```

- Dashboard 不直接依赖其他模块的实现
- 通过 DataBridge 直接读取数据文件
- 各策略模块独立运行，互不干扰

## 快捷操作

2. **运行短线扫描** - 提示命令
